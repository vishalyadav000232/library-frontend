# 📚 Yadav Library – Library Seat Booking & Study Management System

**Yadav Library** is a modern **Library / Study Center Management System** where users can:
✅ explore services  
✅ book study seats  
✅ manage shifts  
✅ track payments  
✅ view availability  
✅ contact the library  

This project is built using **React + Tailwind CSS** (Frontend) and can be extended with **Node/FastAPI + Database** for full backend support.

---

## ✨ Highlights

✅ Modern UI (Tailwind + gradients)  
✅ Responsive layout (Mobile / Tablet / Desktop)  
✅ Seat booking concept + study-friendly design  
✅ Clean reusable components  
✅ Scalable structure for real-world expansion
